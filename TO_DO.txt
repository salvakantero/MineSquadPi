
MINESQUAD Pi
============

EN PROCESO
==========
- adaptar la base de RedPlanet para usar en MineSquad (eliminar lo innecesario)
- menú de inicio

BUGS
====

POR HACER
=========



HECHO
=====
06/04/2024 - paleta pseudopico-8 con 48 colores



Notas
=====

- Isométrico basado en buscaminas (con una versión 2D)

- Tres personajes a elegir:
    Blaze: Velocidad normal, resistencia normal (Es un chico normal)
    Piper: Velocidad alta, resistencia baja (Es una chica)
    Norman: Velocidad baja, resistencia alta (Es un chico fortachón)

- Escenarios (5 niveles cada uno):
     
    Operación Torch (Norte de África):
    Fecha clave: Noviembre de 1942.
    Contexto: Invasión aliada en el norte de África.
    
    Invasión del Día D (Normandía):
    Fecha clave: 6 de junio de 1944.
    Contexto: Desembarco aliado en las playas de Normandía.

    Batalla de las Ardenas:
    Fecha clave: Diciembre de 1944 - enero de 1945.
    Contexto: Ofensiva alemana en la región de las Ardenas durante el invierno.

    Avance Aliado hacia Berlín:
    Fecha clave: 1944-1945 (fases finales de la guerra).
    Contexto: Despeje de obstáculos y avance hacia la capital alemana.

- Objetivo general: 
    Despejar el campo de juego sin activar ninguna mina.

- Campo de juego: 
    Campo de juego rectangular (vertical) dividido en celdas. Hay scroll vertical. 
    Algunas celdas contienen minas, pero el jugador no puede ver dónde están ubicadas.
    El jugador se mueve con joystick o teclado 1 celda cada vez, revelando lo que hay debajo de las adyacentes.
    Si la celda no tiene una mina, mostrará un número que indica cuántas minas hay en las celdas adyacentes.
    Las celdas reveladas vacías se pintan con transparencia verde. Las que tienen números se pintan con transparencia roja.
    El color de los números dependerá del nivel/escenario.
    El jugador puede marcar las celdas que piensa que contienen minas con una bandera o un símbolo similar.
    Para cada mina hay una bandera, así que no puede fallar. A mayor nivel, más minas/banderas.
    El jugador gana el nivel al marcar todas las minas. Pierde si pisa una celda con una mina.
    La estrategia es deducir la ubicación de las minas basándose en los números revelados en las celdas circundantes.

- Objetos:
    Se distribuyen por el mapa aleatoriamente.
    Vida extra: +1
    Munición: permite disparar (+20)
    Gafas rayos X: se destapan 2 filas más de celdas cercanas al player
    Regalos: puntuación extra
        Chuche #1    50
        Chuche #2    75
        Rubí        100
        Diamante    200

- Enemigos:
    Se pasean por el mapa de manera lineal, aleatoria, o persiguen al jugador (pero lentos)
    Dependiendo del nivel, cambian en tipo, número, velocidad y peligrosidad
    Todos mueren pero requieren más o menos disparos

- Detalles técnicos:
    - paleta similar a pico-8, en formato GPL
    - pantallas de fondo de 240x198 en formato PNG
        - logo PlayOnRetro
        - título/intro
        - fondo menú
    - mapas realizados con TILED en formato JSON y tileset común en formato JSON
    - tiles de 16x16 y 16 colores
    - sprites de tamaño variable y 16 colores:
        personajes: 16x16
        objetos: 16x16
        explosiones: 24x24
        disparo: 8x8
    - dos tamaños de fuentes (monócromas):
        grande: 6x16
        pequeña: 3x8
    - tamaño de surfaces (duplicada para escalado):
        WIN_SIZE = 800, 640 # main window size (windowed mode)
        MENU_SCALED_SIZE = 720, 594 # menu size scaled x3 (windowed mode)
        MENU_UNSCALED_SIZE = 240, 198 # menu size (unscaled)
        MAP_SCALED_SIZE = 720, 480 # map size scaled x3 (windowed mode)
        MAP_UNSCALED_SIZE = 240, 160 # map size (unscaled)
        SBOARD_SCALED_SIZE = 720, 114 # scoreboard size scaled x3 (windowed mode)
        SBOARD_UNSCALED_SIZE = 240, 38 # scoreboard size (unscaled)
    - efectos de sonido en formato WAV
    - música en formato OGG
